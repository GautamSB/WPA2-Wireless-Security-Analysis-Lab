# WPA2 Wireless Security Assessment Lab

## 📖 Overview

This repository documents a wireless security assessment performed in a controlled laboratory environment using a TP-Link USB Wi-Fi adapter and the Aircrack-ng suite.

The assessment focuses on wireless reconnaissance, access point discovery, monitor mode operation, IEEE 802.11 frame analysis, WPA2 authentication concepts, and security best practices for wireless networks.

---

# 🎯 Objective

The objectives of this assessment were to:

- Configure a wireless adapter for security testing
- Enable monitor mode
- Discover nearby wireless access points
- Identify wireless network characteristics
- Observe WPA2 authentication behavior
- Capture and analyze IEEE 802.11 management frames
- Understand the WPA2 four-way handshake process
- Evaluate password strength in an authorized lab environment
- Document wireless security observations
- Recommend security hardening measures

# 📚 Methodology

1. Verify wireless adapter.
2. Enable monitor mode.
3. Discover nearby wireless networks.
4. Identify the authorized target access point.
5. Observe beacon, probe, authentication, and association frames.
6. Capture WPA2 authentication traffic.
7. Document wireless network characteristics.
8. Record observations.
9. Recommend security improvements.
